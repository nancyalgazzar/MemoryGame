# 🧠 Interactive Memory Match Game

A polished, fully interactive Memory Match game built from scratch using strict **TypeScript** and modern **CSS3 grid/animations**. This project demonstrates clean code structure, DOM manipulation, state management, and an engaging user experience (UX).

## ✨ Features

*   **TypeScript Architecture:** Built with strict type-safety for robust, bug-free game logic.
*   **Dynamic UI Progress Bar:** A live progress bar updates dynamically as pairs are matched, showing how close the player is to victory.
*   **Immersive Audio:** Audio sound effects for interactions.
*   **Grand Finale Celebration:** High-energy visual and auditory celebration triggers upon successfully matching all cards.
*   **Smart Game Loop:** Includes a real-time move counter, match tracking, and a smart **Shuffle/Reset** mechanism to randomize the board layout on every new round.
*   **Fully Responsive:** Designed using CSS Grid and Flexbox to ensure seamless gameplay across desktops, tablets, and smartphones.

---

## 🛠️ Tech Stack

*   **Language:** TypeScript (compiled to ES6+ JavaScript)
*   **Markup:** HTML5
*   **Styling & FX:** CSS3 
*   **Audio:** HTML5 Audio

## ⚙️ Setup & Installation

To run this project locally, follow these simple steps:

1. **Clone the repository:**
    ```
    git clone https://github.com/nancyalgazzar/MemoryGame.git
    cd memory-game

2. **Install dependencies & Bootstrap:**
    ```
    npm install bootstrap
3. **Compile TypeScript:**
    ```
    npx tsc
4. **Run the project**
    ```
    Open index.html directly in your web browser, or use a local development server extension like VS Code's Live Server.
